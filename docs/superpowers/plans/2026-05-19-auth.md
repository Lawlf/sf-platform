# Auth Implementation Plan (Plan 2)

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Implement passwordless auth (magic link via Resend, Google OAuth PKCE, Apple OAuth PKCE), DB-backed sessions with sliding 30-day expiration, rate limiting (Upstash Redis), CSRF + security headers, account management (deactivation, sessions list/revoke), and the `/entrar`, `/cadastrar`, `/verificar`, `/perfil/seguranca`, `/perfil/conta` user-facing routes.

**Architecture:** All auth flows go through hexagonal use cases under `src/application/use-cases/{auth,account}/`. Domain ports (`UserRepository`, `SessionRepository`, `MagicLinkTokenRepository`, `OauthAccountRepository`, `EmailService`, `OauthProvider`, `RateLimiter`, `RandomGenerator`) live in `src/domain/ports/`. Drizzle repositories, Resend email, Upstash rate limit, Google/Apple OAuth providers live in `src/infrastructure/`. Route handlers in `app/api/auth/**` are thin adapters that call use cases. Session cookie `sf_session` is HttpOnly+Secure+SameSite=Lax. Session IDs stored hashed (sha256) in `sessions.id_hash`; the raw value lives only in the user's cookie.

**Tech Stack:** Next.js 15 Route Handlers + Server Actions, `crypto.subtle` for SHA-256 + random bytes, `jose` for JWT (Apple client secret), Resend for email, React Email for templates, `@upstash/redis` + `@upstash/ratelimit` for rate limit, Zod for input validation.

```ts
function required<K extends keyof Env>(key: K, value: Env[K]): NonNullable<Env[K]> {
  if (value === undefined || value === null || value === "") {
    throw new Error(`Missing required env var: ${String(key)}`);
  }
  return value as NonNullable<Env[K]>;
}

export const DEFAULT_EMAIL_FROM = "onboarding@resend.dev";
```

```ts
import type { Hasher } from "@/domain/ports/services/hasher.service";

export class WebCryptoHasher implements Hasher {
  async sha256Hex(input: string | Uint8Array): Promise<string> {
    const bytes = typeof input === "string" ? new TextEncoder().encode(input) : input;
    const digest = await crypto.subtle.digest("SHA-256", bytes);
    return Array.from(new Uint8Array(digest))
      .map((b) => b.toString(16).padStart(2, "0"))
      .join("");
  }
}
```

```ts
import type { RandomGenerator } from "@/domain/ports/services/random-generator.service";

export class WebCryptoRandomGenerator implements RandomGenerator {
  bytes(length: number): Uint8Array {
    const out = new Uint8Array(length);
    crypto.getRandomValues(out);
    return out;
  }

  urlToken(): string {
    const raw = this.bytes(32);
    return Buffer.from(raw).toString("base64url");
  }

  sixDigitCode(): string {
    const buf = new Uint32Array(1);
    crypto.getRandomValues(buf);
    const n = buf[0] % 1_000_000;
    return n.toString().padStart(6, "0");
  }
}
```

```ts
import { DomainError } from "@/shared/errors";

export class InvalidEmailError extends DomainError {
  readonly code = "INVALID_EMAIL" as const;
}

export class Email {
  private constructor(private readonly value: string) {}

  static from(raw: string): { ok: true; value: Email } | { ok: false; error: InvalidEmailError } {
    const trimmed = (raw ?? "").trim().toLowerCase();
    if (trimmed.length === 0) return { ok: false, error: new InvalidEmailError("Email vazio.") };
    if (trimmed.length > 320) return { ok: false, error: new InvalidEmailError("Email muito longo.") };
    const at = trimmed.indexOf("@");
    if (at < 1 || at === trimmed.length - 1) {
      return { ok: false, error: new InvalidEmailError("Email invalido.") };
    }
    return { ok: true, value: new Email(trimmed) };
  }

  toString(): string {
    return this.value;
  }

  equals(other: Email): boolean {
    return this.value === other.value;
  }
}
```

```ts
        emailVerifiedAt: input.emailVerified ? new Date() : null,
        displayName: input.displayName ?? null,
      })
      .returning();
    return toEntity(rows[0]);
```

```ts
  async findByIdHash(idHash: string): Promise<SessionEntity | null> {

      .from(sessions)
      .where(and(eq(sessions.userId, userId), gt(sessions.expiresAt, new Date())))
      .orderBy(desc(sessions.lastUsedAt));
```

```ts
import type { MagicLinkTokenEntity } from "@/domain/entities/magic-link-token.entity";
import type { MagicLinkTokenRepository } from "@/domain/ports/repositories/magic-link-token.repository";

import { getDb } from "../client";
import { magicLinkTokens } from "../schema/magic-link-tokens.schema";

function toEntity(row: typeof magicLinkTokens.$inferSelect): MagicLinkTokenEntity {
  return {
    tokenHash: row.tokenHash,
    code: row.code,
    email: row.email,
    userId: row.userId,
    expiresAt: row.expiresAt,
    usedAt: row.usedAt,
    attemptCount: row.attemptCount,
    createdAt: row.createdAt,
  };
}

export class DrizzleMagicLinkTokenRepository implements MagicLinkTokenRepository {
  async findByTokenHash(tokenHash: string): Promise<MagicLinkTokenEntity | null> {
    const rows = await getDb().select().from(magicLinkTokens).where(eq(magicLinkTokens.tokenHash, tokenHash)).limit(1);
    return rows[0] ? toEntity(rows[0]) : null;
  }

  async findActiveByEmail(email: string): Promise<MagicLinkTokenEntity | null> {
    const rows = await getDb()
      .select()
      .from(magicLinkTokens)
      .where(
        and(
          eq(magicLinkTokens.email, email.toLowerCase()),
          isNull(magicLinkTokens.usedAt),
          gt(magicLinkTokens.expiresAt, new Date()),
        ),
      )
      .orderBy(desc(magicLinkTokens.createdAt))
      .limit(1);
    return rows[0] ? toEntity(rows[0]) : null;
  }

  async create(input: { tokenHash: string; code: string; email: string; userId: string | null; expiresAt: Date }): Promise<MagicLinkTokenEntity> {
    const rows = await getDb()
      .insert(magicLinkTokens)
      .values({
        tokenHash: input.tokenHash,
        code: input.code,
        email: input.email.toLowerCase(),
        userId: input.userId,
        expiresAt: input.expiresAt,
      })
      .returning();
    return toEntity(rows[0]);
  }

  async markUsed(tokenHash: string): Promise<void> {
    await getDb().update(magicLinkTokens).set({ usedAt: new Date() }).where(eq(magicLinkTokens.tokenHash, tokenHash));
  }

  async incrementAttempts(tokenHash: string): Promise<number> {
    const rows = await getDb()
      .update(magicLinkTokens)
      .set({ attemptCount: sql`${magicLinkTokens.attemptCount} + 1` })

    return rows[0]?.attemptCount ?? 0;

  async deleteExpired(now: Date): Promise<number> {
```

```ts
import { requireUpstashConfig } from "@/infrastructure/config/env";

const redisInstances = new Map<string, Redis>();
const limiterInstances = new Map<string, Ratelimit>();

function getRedis() {
  const cfg = requireUpstashConfig();
  const key = `${cfg.url}:${cfg.token}`;
  if (!redisInstances.has(key)) {
    redisInstances.set(key, new Redis({ url: cfg.url, token: cfg.token }));
  }
  return redisInstances.get(key)!;
}

function getLimiter(window: string, max: number) {
  const key = `${window}:${max}`;
  if (!limiterInstances.has(key)) {
    limiterInstances.set(
      key,
      new Ratelimit({
        redis: getRedis(),
        limiter: Ratelimit.slidingWindow(max, window as `${number} ${"s" | "m" | "h"}`),
        analytics: false,
        prefix: "sf_rl",
      }),
    );
  }
  return limiterInstances.get(key)!;
}

export class UpstashRateLimiter implements RateLimiter {
  async check(key: string, opts: { window: string; max: number }) {
    const result = await getLimiter(opts.window, opts.max).limit(key);
    return {
      ok: result.success,
      remaining: result.remaining,
      resetAt: new Date(result.reset),
    };
  }
}
```

```ts
import type { UserRepository } from "@/domain/ports/repositories/user.repository";
import type { EmailService } from "@/domain/ports/services/email.service";
import type { Hasher } from "@/domain/ports/services/hasher.service";
import type { RandomGenerator } from "@/domain/ports/services/random-generator.service";
import type { RateLimiter } from "@/domain/ports/services/rate-limiter.service";
import type { Clock } from "@/domain/ports/clock.port";

import { Email, InvalidEmailError } from "@/domain/value-objects/email.vo";
import { err, ok, type Result } from "@/shared/errors";
import { MagicLinkEmail } from "@/infrastructure/email/templates/magic-link.email";

export interface RequestMagicLinkDeps {
  users: UserRepository;
  tokens: MagicLinkTokenRepository;
  email: EmailService;
  hasher: Hasher;
  random: RandomGenerator;
  rateLimit: RateLimiter;
  clock: Clock;
  appUrl: string;

export interface RequestMagicLinkInput {
  emailRaw: string;
  ipHash: string;

export type RequestMagicLinkError = InvalidEmailError | RateLimited | AccountDeactivated;

export async function requestMagicLink(
  deps: RequestMagicLinkDeps,
  input: RequestMagicLinkInput,
): Promise<Result<{ tokenHash: string; expiresAt: Date }, RequestMagicLinkError>> {
  const emailResult = Email.from(input.emailRaw);
  if (!emailResult.ok) return err(emailResult.error);
  const email = emailResult.value;

  const emailKey = `magic-link:email:${email.toString()}`;
  const ipKey = `magic-link:ip:${input.ipHash}`;
  const byEmail = await deps.rateLimit.check(emailKey, { window: "1 h", max: 3 });
  if (!byEmail.ok) return err(new RateLimited("Muitas tentativas com este email. Tente em alguns minutos."));
  const byIp = await deps.rateLimit.check(ipKey, { window: "1 h", max: 10 });
  if (!byIp.ok) return err(new RateLimited("Muitas tentativas deste dispositivo."));

  const existing = await deps.users.findByEmail(email.toString());
  if (existing && existing.deactivatedAt) {
    return err(new AccountDeactivated("Conta desativada. Fale com o suporte para reativar."));
  }

  const rawToken = deps.random.urlToken();
  const code = deps.random.sixDigitCode();
  const tokenHash = await deps.hasher.sha256Hex(rawToken);
  const expiresAt = new Date(deps.clock.now().getTime() + 15 * 60 * 1000);
  await deps.tokens.create({
    tokenHash,
    code,
    email: email.toString(),
    userId: existing?.id ?? null,
    expiresAt,
  });

  const html = await render(MagicLinkEmail({ appUrl: deps.appUrl, token: rawToken, code }));
  await deps.email.send({
    to: email.toString(),
    subject: "Seu codigo de acesso ao Sabor Financeiro",
    html,
  });

  return ok({ tokenHash, expiresAt });
}
```

```ts
import type { SessionRepository } from "@/domain/ports/repositories/session.repository";
import type { UserRepository } from "@/domain/ports/repositories/user.repository";
import type { Hasher } from "@/domain/ports/services/hasher.service";
import type { UserEntity } from "@/domain/entities/user.entity";
import { SESSION_COOKIE_NAME } from "@/infrastructure/auth/session-cookie";

export interface CurrentUserDeps {
  sessions: SessionRepository;
  users: UserRepository;
  hasher: Hasher;
  now: Date;
}

export async function loadCurrentUser(deps: CurrentUserDeps): Promise<UserEntity | null> {
  const cookieStore = await cookies();
  const raw = cookieStore.get(SESSION_COOKIE_NAME)?.value;
  if (!raw) return null;
  const idHash = await deps.hasher.sha256Hex(raw);
  const session = await deps.sessions.findByIdHash(idHash);
  if (!session) return null;
  if (session.expiresAt < deps.now) {
    await deps.sessions.delete(idHash);
    return null;
  }
  const user = await deps.users.findById(session.userId);
  if (!user || user.deactivatedAt) return null;
  // sliding expiration: extend on each use
  const newExpires = new Date(deps.now.getTime() + 30 * 24 * 60 * 60 * 1000);
  await deps.sessions.touch(idHash, newExpires, deps.now);
  return user;
}
```

```ts
function decodeJwtPayload(jwt: string): Record<string, unknown> {
  const parts = jwt.split(".");
  if (parts.length < 2) throw new Error("Invalid JWT");
  const payload = parts[1].replace(/-/g, "+").replace(/_/g, "/");
  const padded = payload + "=".repeat((4 - (payload.length % 4)) % 4);
  return JSON.parse(Buffer.from(padded, "base64").toString("utf-8"));
```

```ts
  }

  async exchangeCode(input: { code: string; codeVerifier: string }): Promise<OauthProfile> {
    const cfg = requireAppleOauthConfig();
    const env = loadEnv();
    const clientSecret = await this.buildClientSecret();
    const body = new URLSearchParams({
      grant_type: "authorization_code",
      code: input.code,
      client_id: cfg.clientId,
      client_secret: clientSecret,
      redirect_uri: `${env.NEXT_PUBLIC_APP_URL}/api/auth/apple/callback`,
      code_verifier: input.codeVerifier,
    });
    const tokenResp = await fetch("https://appleid.apple.com/auth/token", {
      method: "POST",
      headers: { "Content-Type": "application/x-www-form-urlencoded" },
      body,
    });
    if (!tokenResp.ok) throw new Error(`Apple token exchange failed: ${tokenResp.status}`);
    const tokens = (await tokenResp.json()) as { id_token: string };
    const payload = decodeJwtPayload(tokens.id_token);
    return {
      provider: "apple",
      providerUserId: String(payload.sub),
      email: String(payload.email),
      emailVerified: Boolean(payload.email_verified ?? true),
      displayName: null,
    };
  }

  private async buildClientSecret(): Promise<string> {
    const cfg = requireAppleOauthConfig();
    const privateKey = await importPKCS8(cfg.privateKey.replace(/\\n/g, "\n"), "ES256");
    const now = Math.floor(Date.now() / 1000);
    return new SignJWT({})
      .setProtectedHeader({ alg: "ES256", kid: cfg.keyId })
      .setIssuer(cfg.teamId)
      .setIssuedAt(now)
      .setExpirationTime(now + 60 * 60 * 24 * 30) // 30 days; Apple max is 6 months
      .setAudience("https://appleid.apple.com")
      .setSubject(cfg.clientId)
      .sign(privateKey);
  }
}

function decodeJwtPayload(jwt: string): Record<string, unknown> {
  const parts = jwt.split(".");
  if (parts.length < 2) throw new Error("Invalid JWT");
  const payload = parts[1].replace(/-/g, "+").replace(/_/g, "/");
  const padded = payload + "=".repeat((4 - (payload.length % 4)) % 4);
  return JSON.parse(Buffer.from(padded, "base64").toString("utf-8"));
}
```

## Risks and rollback notes

- **Resend domain not verified.** Magic links cannot send until SPF/DKIM/DMARC verify. Block on Task 28 manual smoke until done.
- **Apple OAuth complexity.** P8 private key handling, JWT client secret rotation, form_post callback mode. If unable to test in this environment, mark the Apple slice DONE_WITH_CONCERNS and defer manual validation.
- **Sliding session expiration.** Every protected route bumps the expiration. If a session is open in two tabs, the second tab will refresh the cookie. Cookie max-age stays 30 days from latest use.
- **Account deactivation does NOT cascade-delete.** Per LGPD design choice, user data remains. The `sessions` cascade delete is still active because session FK uses `ON DELETE CASCADE` against `users`, but we explicitly do NOT delete the user row.
- **Email verification flow.** Currently, magic-link verify marks the user verified. OAuth also marks verified if provider says so. Until SPF/DKIM verifies, do not allow OAuth-Google email matching to auto-link (Task 22 returns `OauthAccountLinkRequiresVerification` instead).
- **Rate limit fallback.** If Upstash is down, our adapter throws, and `RequestMagicLink` returns a 500-style error from the route handler. Consider a feature-flag fallback (allow without rate-limit) in Plan 3 if reliability is an issue.
